## Correctif installateur Windows

La version **1.5.0** produisait l'erreur Inno Setup :
*"The setup files are corrupted, please obtain a new copy of the program"*.

**Utilisez cette version 1.5.1** — installateur reconstruit et testé localement.

### Installation
1. Téléchargez `TabernacleERP-Setup-1.5.1.exe`
2. Exécutez l'installateur (droits administrateur)
3. Lancez Tabernacle de la Moisson ERP depuis le menu Démarrer
