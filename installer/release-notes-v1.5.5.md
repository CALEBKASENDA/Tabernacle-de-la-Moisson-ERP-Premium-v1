## Tabernacle ERP Premium v1.5.5 — mode desktop Windows

### Changements
- **Fenêtre application dédiée** (Edge app mode) : plus d'ouverture navigateur classique
- **Local First hybride** : données locales + déploiement cloud/VPS optionnel
- **Node 24** embarqué (compatible better-sqlite3)

### Installation
1. Désinstallez l'ancienne version si besoin
2. Installez `TabernacleERP-Setup-1.5.5.exe`
3. Lancez depuis le menu Démarrer

### Déploiement cloud (VPS / hébergeur)
Le même code peut tourner sur un serveur avec Docker ou Node.js — voir `docker-compose.yml` et `deploy/`.
